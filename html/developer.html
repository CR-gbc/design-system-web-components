Development Setup

git clone https://apps.gcpe.gov.bc.ca/bitbucket/scm/~sturple/bcgov-starter-kit.git
mv bcgov-starter-kit my-project
cd my-project
npm install
npm run start

Start Dev Server

npm run start

Build Prod Version

npm run build
